# خصائص قارات العالم (World Continents Characteristics)

تطبيق تفاعلي ثلاثي الأبعاد يعرض مجسم الكرة الأرضية وخريطة مسطحة، ويتيح للمستخدم استكشاف القارات والتعرف على أهم خصائصها الجغرافية مع دعم المؤثرات الصوتية والنطق الصوتي للمعلومات.

## التشغيل محلياً (Run Locally)

**المتطلبات الأساسية:** `Node.js`

1. تثبيت الحزم المطلوبة (Install dependencies):
   ```bash
   npm install --legacy-peer-deps
   ```

2. تشغيل التطبيق (Run the app):
   ```bash
   npm run dev
   ```
